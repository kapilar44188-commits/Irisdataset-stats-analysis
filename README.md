 What's Covered

1. Descriptive Statistics

Summary of mean, standard deviation, min/max, and percentiles for all 4 features across 3 species.

2. Distribution Plots

Histograms and box plots for each feature, split by species — so you can see the data before testing it.

3. Normality Tests — Shapiro–Wilk

Tests whether each feature follows a normal distribution per species.
Required before running t-tests.

4. T-Tests — Welch's Independent Samples

Two comparisons:


Setosa vs Versicolor — petal length
Versicolor vs Virginica — sepal width


Uses Welch's version (does not assume equal variance between groups).

5. Confidence Intervals (95%)

Estimates the plausible range for the true mean petal length of each species.

6. Chi-Square Test

Tests whether petal size category (Small / Medium / Large) is independent of species — or strongly linked to it.
